# spacecruiser
Discordサーバー、sanchezyspace管理用Bot

# 導入
1. 環境変数を以下のように記述
    ```
    DISCORD_TOKEN = {your discord app token}
    GUILD_ID = {your server id}
    ```
1. npm install
    ```
    $ npm install
    ```

2. ビルド
    ```
    $ npm run build
    ```
3. 実行
    ```
    $ npm start
    ```
